🧊 ALFA COLDRES

Sistema simples de login e cadastro feito em Python.

📌 Sobre o projeto

O ALFA COLDRES é um programa de terminal que simula um sistema básico de autenticação, permitindo que o usuário faça login ou cadastro.

⚙️ Funcionalidades
	•	Login com usuário e senha
	•	Validação de acesso
	•	Cadastro de novo usuário
	•	Confirmação de senha no cadastro

🧠 Lógica utilizada

O sistema utiliza:
	•	Estruturas condicionais (if, elif, else)
	•	Entrada de dados com input()
	•	Comparações de strings

💻 Tecnologias
	•	Python

▶️ Como executar
	1.	Instale o Python no seu computador
	2.	Copie o código para um arquivo .py
	3.	Execute no terminal.
